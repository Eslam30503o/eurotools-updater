# 🔧 إدارة الأدوات - دليل شامل

دليل مفصل لإدارة الأدوات في Euro Tools Code Manager، من الإضافة إلى التنظيم والبحث.

## 🎯 نظرة عامة

نظام إدارة الأدوات هو القلب النابض لـ Euro Tools Code Manager، حيث يمكنك تنظيم وإدارة جميع أدواتك الصناعية بكفاءة عالية.

## 🆕 إضافة الأدوات {#adding-tools}

### الطريقة الأساسية

#### 1. إضافة أداة واحدة
```
📋 خطوات إضافة أداة جديدة:
1. انقر على زر "إضافة أداة جديدة" 🆕
2. املأ البيانات المطلوبة
3. اختر الفئة المناسبة
4. احفظ الأداة 💾
```

### الطرق المتقدمة للإضافة

#### 2. الإضافة المجمعة (Batch Import)
```
📊 استيراد مجمع من Excel:
1. حمل قالب Excel [📥 تحميل القالب]
2. املأ البيانات في الجدول
3. احفظ الملف واستورده [📤 استيراد]
4. راجع البيانات وأكد الاستيراد ✅
```

#### 3. الإضافة من الباركود
```
📱 إضافة بالباركود:
1. انقر على "مسح باركود" 📷
2. وجه الكاميرا نحو الباركود
3. سيتم ملء البيانات تلقائياً
4. راجع وأكمل البيانات المفقودة
```

#### 4. النسخ من أداة موجودة
```
📋 نسخ أداة موجودة:
1. حدد الأداة المراد نسخها
2. انقر على "نسخ أداة" 📋
3. عدل البيانات المختلفة
4. احفظ الأداة الجديدة 💾
```

## 📋 قوالب الأكواد {#code-templates}

#### أنماط الأكواد المدعومة
```
📝 أنماط الأكواد:
├── ABC-#### (مثل: CT-0001)
├── ABC##-### (مثل: CT01-001)  
├── ABC-##-#### (مثل: CT-01-0001)
├── ####-ABC (مثل: 0001-CT)
└── ABC####ABC (مثل: CT0001DRL)

المتغيرات المتاحة:
• # = رقم تسلسلي
• A = حرف كبير
• a = حرف صغير
• Y = السنة (4 أرقام)
• M = الشهر (2 رقم)
• D = اليوم (2 رقم)
```
